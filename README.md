# dados-renda-variavel

Script que faz scrape em site de dados de FIIs.

## Como rodar

```
pip3 install -r requirements.txt
python3 main.py
```

## Para desenvolver

```
sudo apt install python-venv
python3 -m venv venv
pip3 install -r requirements.txt
```

- Exportar para planilha
